# Inglés II - Unidad 1 - Ejercicios Interactivos

Este proyecto permite completar los ejercicios de la Unidad 1 de Inglés II (UTN FRLP) en formato web, replicando el PDF original y permitiendo exportar las respuestas en PDF para su entrega.

## Estructura
- index.html: Página principal con los ejercicios.
- styles.css: Estilos visuales.
- script.js: Lógica para manejo de respuestas y exportación a PDF.

## Uso
1. Completa los ejercicios en la web.
2. Haz clic en "Exportar a PDF" para descargar tus respuestas.
3. Entrega el PDF generado según las indicaciones de la cátedra.

## Despliegue
- El sitio está preparado para ser alojado en GitHub Pages.
